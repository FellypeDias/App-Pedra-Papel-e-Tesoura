# Pedra Papel e Tesoura

Este e um aplicativo simples de Jokenpo que desenvolvi para Android. O objetivo principal foi praticar logica de programacao em Java e entender como funciona a interacao entre o layout XML e o codigo.

Como funciona:
O usuario escolhe uma das tres opcoes (pedra, papel ou tesoura) clicando na imagem correspondente. O aplicativo gera uma escolha aleatoria para o robo e compara com a escolha do usuario para definir quem ganhou, perdeu ou se houve empate.

Tecnologias utilizadas:
- Java
- Android Studio
- Layout em XML (ConstraintLayout)

Como rodar o projeto:
1. Baixe ou clone o repositorio.
2. Abra a pasta no Android Studio.
3. Execute o app em um emulador ou no seu proprio celular.
